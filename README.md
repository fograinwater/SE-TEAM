#### 牧场智能巡逻车

​	担任技术负责人，领导团队研发智能巡逻车。负责项目规划、系统设计和团队管理。在项目初期，我与团队一起进行了市场调研和需求分析，确定了巡逻车需要具备的功能和性能要求。随后，我主导了系统架构设计，选择了合适的传感器、控制系统和通信模块，并与硬件团队密切合作，确保系统的稳定性和可靠性。在开发过程中，我积极参与了巡逻车的自主导航算法的设计和开发，并负责解决遇到的技术难题和系统故障。最终，我们成功研发出一款稳定可靠的巡逻车原型，并进行了现场测试。该巡逻车能够实现自主巡航、动物健康监测和异常报警等功能，受到了牧场管理方的高度认可和好评。
